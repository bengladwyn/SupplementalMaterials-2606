# SupplementalMaterials-2606.14853
Supplemental material for *The
limits of lattice inflation: a cautionary tale* (Barker, Gladwyn, Zell), http://arxiv.org/abs/2606.14853

### Contents

- `Analytics` - Contains scripts for the USR and SR model which calculate the analytic evolution of the models and produce the inputs for CosmoLattice.
- `CLmodels` - the model and input files used to run CosmoLattice.
- `CLPlotting` - Notebooks to plot the CosmoLattice output (saved in `CLOutput` and `ModifiedCLOutput`) compared to the analytic expectations (produced by the scripts in `Analytics`).
- `PerformantMS` - Standalone reproduction kit for the Mukhanov-Sasaki solver benchmark of the appendix.
- `CLOutput` - Output from CosmoLattice evolution for the three models
- `ModifiedCLOutput` - Output from CosmoLattice evolution with 1st order perturbations for the three models
